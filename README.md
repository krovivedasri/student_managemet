# Student Result Management System

This is a simple console-based Student Result Management System created using Python.
It allows the user to manage student records such as adding, updating, deleting, and viewing details.

The system also calculates total marks, percentage, and grade automatically based on student marks.

### Features:

* Add new student details
* View individual student data
* Update marks and phone number
* Delete student record
* Display all students
* Save data to CSV file using pandas
* View summary statistics (average percentage and grade distribution)

### How it works:

* User selects options from menu
* Student data is stored using dictionary
* Marks are validated and processed
* Grade is calculated based on percentage
* Data can be saved into CSV file
* Summary shows overall performance

### Technologies Used:

* Python
* Pandas
* Collections (Counter)

### Output:

* Displays student details
* Calculates percentage and grade
* Saves records in CSV file
* Shows average and grade distribution

### Note:

* Marks should be between 0 and 100
* Phone number should contain only digits
* Each student is identified using Roll Number
